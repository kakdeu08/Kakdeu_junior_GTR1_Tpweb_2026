Intrduction

Ce mini-site a été créer dans le cadre du cours de Tp programmation web

Il permet de découvrir la création de plusieurs pages html reliées entre elles .

Objectif du mini-site

L'Objectif est d'apprendre à structurer un petit projet web avec plusieurs fichiers

Pages disponibles

  • La page Accueil présente le contexte général du mini-site.
  • La page présentation contient les informations personnelles de l'étudiant.
  • La page Contact contient les informations de contactde l'étudiant
